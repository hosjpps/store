# Anime Store — магазин манги, манхвы и ранобэ

Веб-витрина онлайн-магазина с каталогом манги, манхвы и ранобэ: фильтры, корзина, избранное, заказы и админ-панель. Построена на Next.js 15 и React 19.

## Возможности

- Каталог товаров с фильтрами по жанру, типу, цене и рейтингу, сортировкой, поиском и пагинацией
- Карточка товара с отзывами и оценками, быстрый просмотр (Quick View) и блок «Недавно просмотренные»
- Корзина с изменением количества и применением промокодов (процентные и фиксированные скидки)
- Избранное с сохранением между сессиями
- Личный кабинет: регистрация, вход, история заказов
- Оформление заказа и просмотр статусов (в обработке, отправлен, доставлен и т. д.)
- Админ-панель: управление товарами, заказами, пользователями, отзывами и настройками
- Мобильный UX: нижняя навигация, pull-to-refresh, адаптивная вёрстка, светлая и тёмная темы

## Стек

- **Next.js 15** (App Router), **React 19**, **TypeScript**
- **Tailwind CSS 4** + компоненты на базе **Radix UI** (shadcn/ui)
- **Zustand** для управления состоянием (корзина, избранное, заказы, темы) с персистентностью
- **React Hook Form** + **Zod** для форм и валидации
- **Recharts** для графиков в админке, **Lucide** для иконок
- Данные каталога, отзывов и пользователей — в локальных JSON-файлах (мок-слой без бэкенда)

## Структура

```
app/          — маршруты App Router
  page.tsx        — главная страница с каталогом и фильтрами
  product/[id]    — страница товара
  cart/           — корзина
  favorites/      — избранное
  account/        — личный кабинет и авторизация
  order/[id]      — страница заказа
  admin/          — админ-панель (товары, заказы, пользователи, отзывы, настройки)
components/    — UI-компоненты (header, product-card, отзывы, фоновая анимация и др.)
  ui/             — переиспользуемые компоненты на Radix
lib/store.ts  — глобальное состояние на Zustand
hooks/        — пользовательские хуки (auth, toast, mobile)
data/         — мок-данные: products.json, reviews.json, users.json
types/        — общие TypeScript-типы
```

## Запуск

```bash
npm install
npm run dev
```

Приложение будет доступно на http://localhost:3000.

Сборка для продакшена:

```bash
npm run build
npm start
```
